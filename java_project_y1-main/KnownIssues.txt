Change #9 - Collage: extracted createLecturer() factory method [SRP / OCP]
File: Collage.java
Description: Extracted private createLecturer(name, id, degree, degName, salary,
articleSize, academy, artSet) method. The if/else chain that decided which
Lecturer subclass to instantiate was moved out of lecturerToCollage() into
this dedicated private method. lecturerToCollage() now only orchestrates.
Principle: SRP (one method, one job) and OCP (creation logic is isolated;
adding a new degree type means changing only createLecturer).

Change #10 - Collage: extracted createCommittee() factory method [SRP / OCP]
File: Collage.java
Description: Extracted private createCommittee(comName, headLec, lecLevel) method.
The if/else chain selecting the Committee generic type was moved out of
committeeToCollage() into this method. committeeToCollage() now only orchestrates.
Principle: SRP and OCP, same rationale as Change #9.

Change #11 - Collage: removed dead static Scanner field [SRP]
File: Collage.java
Description: Removed static Scanner scn field that was declared but never used
inside Collage. Main.java already owns its own Scanner for all user input.
Principle: SRP - Collage should not own I/O concerns.
Also removed the now-unused import java.util.Scanner.

Change #12 - Lecturer: fixed equals() unsafe cast [LSP]
File: Lecturer.java
Description: Added instanceof check before casting in equals(Object o).
The original code cast directly without checking, which would throw a
ClassCastException when comparing a Lecturer to a non-Lecturer object.
Principle: LSP - a class must honor the contract of Object.equals(),
which must handle any Object without throwing.

Change #13 - Department: fixed equals() unsafe cast [LSP]
File: Department.java
Description: Same instanceof fix as Change #12 applied to Department.equals().
Also removed the private lecInDep(Lecturer) method that was declared but
never called anywhere in the codebase.
Principle: LSP (equals contract), ISP (remove unused surface).

Change #14 - Committee: removed dead lecSize field, fixed equals() [LSP / ISP]
File: Committee.java
Description: Removed private int lecSize field. getLecSize() already derived
its value from committeeMembers.size(); the field was set to 0 in the constructor
and never incremented, making it permanently 0. The equals() method included
lecSize == committee.lecSize which was always 0 == 0 - a meaningless check that
gave a false sense of correctness in the equals contract.
Fixed equals() to use instanceof instead of getClass() comparison for proper
substitutability, and removed the lecSize term from both equals() and hashCode().
Principle: LSP (correct equals contract), ISP (remove dead state).

Change #15 - Lecturer: removed dead comSize field [ISP]
File: Lecturer.java
Description: Removed private int comSize field. It was declared and initialized
to 0 but never read or modified anywhere in the class or the project.
Principle: ISP - classes should not carry state that serves no interface or consumer.

Change #16 - SortComByArc: made sumArticles() private, renamed [SRP]
File: SortComByArc.java
Description: Renamed public runOnArray() to private sumArticles(). A Comparator
exposes one contract: compare(). The helper that sums articles is an internal
detail and must not leak into the public API of the class.
Principle: SRP - the class has one public responsibility (comparing); the helper
is an implementation detail.

Change #17 - StringBuilder adoption across all toString/display methods [quality]
Files: Lecturer.java, Department.java, Committee.java, Doctor.java, Collage.java
Description: Replaced string concatenation inside Iterator loops with StringBuilder
across all display and toString methods. String concatenation inside loops creates
a new String object on every iteration; StringBuilder accumulates into one buffer.
All loops still use explicit Iterator as required by the project.
